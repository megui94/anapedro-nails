SITE (Template Agenda de Beleza) — com página "Sobre"

Ficheiros:
- index.html   (agenda fechada)
- sobre.html   (página Sobre)
- styles.css   (design)
- logo.jpg     (logo)
- waves.svg    (fundo estilo 'agenda de beleza')
- netlify.toml (config Netlify)

PUBLICAR NO NETLIFY
1) Descompacta o ZIP.
2) Netlify → Add new site → Deploy manually
3) Arrasta TODOS os ficheiros para o deploy (inclui sobre.html).

EDITAR INFORMAÇÕES (rápido)
- Instagram: procura "instagram.com/anapedro.nails" em index.html e sobre.html
- Localização/Horários: em sobre.html procura "[Escreve aqui ...]"
- Textos/Serviços: edita diretamente nos ficheiros HTML


FOTOGRAFIA NA PÁGINA "SOBRE"
- Para adicionares a tua foto, coloca uma imagem na mesma pasta do site com o nome:
  minha-foto.jpg
- Depois faz novo deploy no Netlify (arrastar os ficheiros atualizados).
